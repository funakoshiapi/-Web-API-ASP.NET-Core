# What is it for?

- Web API for Mapping web application that renders polygons representing tax areas of states, cities, couties or districts in the United States

# Tecnologies used 
 - API-ASP.NET-Core
 - MongoDb
 - Entity Framework
 - SQL Server Management Studio
 - GoogleMaps Geocoding API
